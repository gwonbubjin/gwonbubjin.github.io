# 국립중앙박물관 홍보 웹앱 폴더 구조

## 최종 구조 (단일 페이지)

```
museum-website/
├── index.html                 # 메인 페이지 (모든 섹션 포함)
├── css/
│   └── main.css               # 모든 스타일 (반응형 포함)
├── js/
│   └── main.js                # 모든 JavaScript 기능
└── images/                    # 이미지 파일 (선택사항)
    ├── logo/
    ├── exhibitions/
    ├── events/
    └── collections/
```

## 포함된 섹션

1. **소개** (`#about`) - 박물관 소개 및 하이라이트
2. **관람정보** (`#visit-info`) - 관람시간, 입장료, 문의, 주차안내
3. **전시** (`#exhibitions`) - 현재 진행 중인 전시
4. **문화행사** (`#events`) - 문화 체험 프로그램
5. **컬렉션 검색** (`#collection-search`) - 소장품 검색 및 필터링
6. **공지** (`#notice`) - 공지사항 목록
7. **오시는길** (`#location`) - 위치 및 교통편 안내

## 주요 기능

- ✅ 반응형 웹 디자인 (모바일/태블릿/데스크톱)
- ✅ 부드러운 스크롤 네비게이션
- ✅ 모바일 햄버거 메뉴
- ✅ 컬렉션 검색 기능
- ✅ 카테고리 필터링
- ✅ 스크롤 애니메이션

## GitHub Pages 호스팅 고려사항

1. **index.html**은 루트에 있어야 함 ✅
2. 모든 경로는 상대 경로로 작성 ✅
3. 대소문자 구분 주의 (일부 서버는 구분함)
4. 이미지 최적화 필수 (로딩 속도)
5. GitHub 저장소 Settings > Pages에서 브랜치 선택
