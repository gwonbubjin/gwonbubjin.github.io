About Me

어린 시절, 뒤죽박죽인 숫자들을 엑셀(Excel)에 넣어 질서 정연하게 정리하고, 그 안에서 의미를 찾아내는 과정에 큰 흥미를 느꼈습니다. 이러한 데이터에 대한 호기심은 고등학교 시절 프로그래밍을 만나며 '직접 만드는 즐거움'으로 확장되었습니다.

저는 단순히 기능을 구현하는 것을 넘어, 데이터 분석과 웹 개발 두 가지 영역을 모두 사랑하는 개발자입니다. React와 TypeScript로 사용자가 머물고 싶은 직관적인 UI/UX를 구현하는 동시에, Python과 SQL을 통해 서비스 이면에 숨겨진 데이터의 가치를 발굴하는 데에도 열정이 넘칩니다.

데이터가 흐르는 구조를 이해하고, 이를 가장 효과적으로 사용자에게 전달하는 '데이터를 볼 줄 아는 웹 & 앱 개발자'로서 성장하고 있습니다.

Troubleshooting

ISSUE

Chrome Extension CSP(보안 정책) 위반 에러

문제: 팝업 HTML 버튼에 인라인 이벤트를 작성했더니, 보안 에러가 발생하며 작동하지 않음.

해결: Manifest V3의 보안 정책상 인라인 스크립트가 금지됨을 확인. 로직을 별도 JS 파일로 분리하고 addEventListener로 이벤트를 연결하여 해결.

#ChromeExtension #ManifestV3 #CSP
ISSUE

JSP 데이터 전송 시 한글 깨짐 현상

문제: 회원가입 폼이나 게시판에 한글 데이터를 입력해서 서버로 보내면 문자가 깨져서 저장되는 현상 발생.

해결: Tomcat 서버의 기본 인코딩 문제임을 파악. 데이터를 받기 전 인코딩 코드를 추가하여 한글이 정상적으로 처리되도록 수정.

#JSP #Tomcat #Encoding
ISSUE

PWA 업데이트 내용 미반영 이슈

문제: 배포 후 코드를 수정했는데 사용자의 브라우저에서 이전 캐시 화면만 계속 보이는 현상.

해결: service-worker.js의 버전을 갱신하고, 활성화될 때 기존 캐시를 삭제하는 로직을 추가하여 해결.

#PWA #ServiceWorker #Cache
